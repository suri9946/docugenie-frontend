# DocuGenie Frontend

Production-ready React + Vite frontend for DocuGenie AI Document Generator.

## Setup

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Build

```bash
npm run build
npm run preview
```

## Environment

Copy `.env.example` to `.env.local` and update API URL if needed:

```
VITE_API_URL=http://localhost:5000
```

## Features

- Clean two-column layout
- Real-time document generation
- Preview with lock/unlock
- Download integration
- Responsive design
- Error handling
- Modern SaaS UI
